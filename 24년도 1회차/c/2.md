## 🧮 문제 2 - C언어 삼항 연산자와 시프트 연산

> 📌 **문제 설명**
> 다음 C 코드를 실행했을 때 출력되는 값을 작성하시오.

---

### 📄 문제 코드

```c
#include <stdio.h>

int main() {
    int v1 = 0, v2 = 35, v3 = 29;

    if (v1 > v2 ? v2 : v1) {
        v2 = v2 << 2;
    } else {
        v3 = v3 << 2;
    }

    printf("%d", v2 + v3);
}
```

---

### 🔍 문제 해설

#### ✅ 1. 삼항 연산자 해석

```c
v1 > v2 ? v2 : v1
```

* `v1 = 0`, `v2 = 35`
* `v1 > v2`는 `false` → 조건식 결과는 `v1 = 0`
* `if (0)` 이므로 `false` → `else` 문 실행

---

#### ✅ 2. 실행 흐름

```c
else {
    v3 = v3 << 2;
}
```

* `v3 = 29`
* `v3 << 2`는 **왼쪽으로 2비트 시프트** → `29 × 2⁴ = 29 × 4 = 116`

```c
v2 = 35;
v3 = 116;
```

---

#### ✅ 3. 출력 결과 계산

```c
v2 + v3 = 35 + 116 = 151
```

---

### ✅ 최종 출력값

```
151
```

---

### 💡 개념 요약

| 개념           | 설명                                          |
| ------------ | ------------------------------------------- |
| 삼항 연산자       | `condition ? A : B` → 조건이 true면 A, false면 B |
| 시프트 연산자 `<<` | 왼쪽으로 비트 이동 (n비트 이동 → 값 × 2ⁿ)                |
| `if (0)`     | `false`로 간주                                 |

---

### 📘 코드 흐름 요약

```c
v1 = 0
v2 = 35
v3 = 29

조건: v1 > v2 ? v2 : v1  ➜  0
→ if (0) → else 실행

v3 = 29 << 2 → 116
v2 = 35
출력: v2 + v3 = 151
```
